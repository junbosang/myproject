## 一、项目概述

### 1.1 项目定位
cnAgentOS 是一个 **AI智能瞭望与智能文数系统**，基于 Tornado MVC 架构，提供用户侧智能问答和管理侧数字员工管理、数据采集、数据可视化等核心功能。

### 1.2 核心目标
- 为用户提供基于大模型的自然语言问答服务
- 为管理员提供数字员工管理、数据采集、数据仓库管理等后台能力
- 构建可配置、可扩展的本地化私有AI服务平台
- 提供数智大屏可视化展示能力

-- 数字员工主表
CREATE TABLE IF NOT EXISTS digital_workers (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL UNIQUE,           -- 员工名称
    description TEXT,                     -- 员工描述
    category TEXT NOT NULL,               -- 分类: data/weather/news/music/movie
    status INTEGER DEFAULT 1,             -- 状态: 0停用 1启用
    model_config TEXT,                    -- 模型配置JSON
    data_source_config TEXT,              -- 数据源配置JSON
    created_at TEXT DEFAULT(datetime('now')),
    updated_at TEXT DEFAULT(datetime('now'))
);

-- 员工调用日志表
CREATE TABLE IF NOT EXISTS worker_call_logs (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    worker_id INTEGER NOT NULL,           -- 关联数字员工ID
    user_id INTEGER,                      -- 调用用户ID
    question TEXT NOT NULL,               -- 用户提问
    answer TEXT,                          -- 员工回答
    response_time REAL,                   -- 响应时间(秒)
    status INTEGER DEFAULT 1,             -- 状态: 0失败 1成功
    created_at TEXT DEFAULT(datetime('now')),
    FOREIGN KEY (worker_id) REFERENCES digital_workers(id)
);
```

-- 模型配置表
CREATE TABLE IF NOT EXISTS llm_models (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL UNIQUE,            -- 模型名称
    type TEXT NOT NULL,                   -- 类型: local/remote
    api_url TEXT,                         -- API地址
    api_key TEXT,                         -- API密钥
    model_params TEXT,                    -- 模型参数JSON(temperature, max_tokens等)
    is_stream INTEGER DEFAULT 0,          -- 是否支持流式: 0否 1是
    status INTEGER DEFAULT 1,             -- 状态: 0停用 1启用
    is_default INTEGER DEFAULT 0,         -- 是否默认模型: 0否 1是
    created_at TEXT DEFAULT(datetime('now')),
    updated_at TEXT DEFAULT(datetime('now'))
);

-- Token使用统计表
CREATE TABLE IF NOT EXISTS llm_token_stats (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    model_id INTEGER NOT NULL,            -- 关联模型ID
    user_id INTEGER,                      -- 用户ID
    prompt_tokens INTEGER DEFAULT 0,      -- 输入Token数
    completion_tokens INTEGER DEFAULT 0,  -- 输出Token数
    total_tokens INTEGER DEFAULT 0,       -- 总Token数
    question TEXT,                        -- 提问内容
    created_at TEXT DEFAULT(datetime('now')),
    FOREIGN KEY (model_id) REFERENCES llm_models(id)
);
```

## 三、非功能需求

### 3.1 性能需求

| 指标项 | 要求 | 说明 |
|--------|------|------|
| 页面响应时间 | < 2秒 | 普通页面加载和接口响应 |
| 问答响应时间 | < 10秒 | 从用户提问到开始输出回答 |
| 流式首字时间 | < 3秒 | 流式响应时首个字符输出时间 |
| 并发用户数 | 支持 50+ 并发 | 本地部署环境下的并发处理能力 |
| 数据库查询 | < 1秒 | 常规数据库查询响应时间 |

### 3.2 安全需求

| 安全项 | 要求 | 实现方式 |
|--------|------|----------|

### 4.3 权限数据库表设计

```sql
-- 角色表
CREATE TABLE IF NOT EXISTS roles (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL UNIQUE,            -- 角色名称
    code TEXT NOT NULL UNIQUE,            -- 角色编码
    description TEXT,                     -- 角色描述
    status INTEGER DEFAULT 1,             -- 状态: 0停用 1启用
    created_at TEXT DEFAULT(datetime('now'))
);

-- 权限表
CREATE TABLE IF NOT EXISTS permissions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,                   -- 权限名称
    code TEXT NOT NULL UNIQUE,            -- 权限编码(如 user:create)
    module TEXT,                          -- 所属模块
    description TEXT,
    created_at TEXT DEFAULT(datetime('now'))
);

-- 角色权限关联表
CREATE TABLE IF NOT EXISTS role_permissions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    role_id INTEGER NOT NULL,
    permission_id INTEGER NOT NULL,
    FOREIGN KEY (role_id) REFERENCES roles(id),
    FOREIGN KEY (permission_id) REFERENCES permissions(id),
    UNIQUE(role_id, permission_id)
);

-- 用户角色关联表
CREATE TABLE IF NOT EXISTS user_roles (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    role_id INTEGER NOT NULL,
    FOREIGN KEY (user_id) REFERENCES users(id),
    FOREIGN KEY (role_id) REFERENCES roles(id),
    UNIQUE(user_id, role_id)
);
```
